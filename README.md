# LessonTask 1.0 - 字节学习任务单编辑器

一个功能完整的、生产级别的交互式学习任务单编辑器，可以创建包含多种题型的HTML学习单。

## 核心功能

### 📝 编辑器功能
- **可视化编辑**: 左侧组件库 + 中间所见即所得画布
- **丰富组件库**:
  - 基础组件：文档标题、正文段落、二列/三列布局
  - 题目工具：单选题、多选题、填空题、问答题
  - 高级组件：盲盒抽奖、嵌入HTML代码
- **实时编辑**: 点击即可配置每个组件
- **灵活排序**: 上移/下移/删除组件
- **自动保存**: 每30秒自动保存至云端

### 🎓 学生端体验
- **独立HTML文件**: 生成的学习单是完全独立的HTML文件，无需服务器即可使用
- **即时反馈**:
  - 单选题：点击选项立即显示对错
  - 多选题：提交后显示批改结果
  - 填空题：失焦时自动判断对错
  - 问答题：可查看参考答案
- **交互动画**: 盲盒翻转效果、答题反馈动画
- **响应式设计**: 完美支持手机、平板、电脑

### 👤 用户系统
- Supabase Auth认证
- 用户可管理自己的学习单草稿
- 安全的行级访问控制

## 技术栈

- **前端**: React 18 + TypeScript + Vite
- **样式**: Tailwind CSS（自定义设计系统）
- **路由**: React Router v7
- **数据库**: Supabase (PostgreSQL + RLS)
- **图标**: Lucide React

## 使用说明

### 创建学习单

1. 注册并登录账号
2. 点击"新建学习单"按钮
3. 从左侧组件库中选择需要的组件
4. 在弹出的配置窗口中填写内容
5. 组件会自动添加到画布中

### 编辑组件

- **悬停显示操作**: 鼠标悬停在组件上时显示编辑按钮
- **上移/下移**: 调整组件顺序
- **编辑**: 重新打开配置窗口修改内容
- **删除**: 移除组件（需确认）

### 生成学习单

1. 点击顶部"生成学习单"按钮
2. 系统会自动保存并生成HTML文件
3. 下载的HTML文件可直接分发给学生使用
4. 学生双击打开即可作答，无需任何服务器或网络连接

### 题型说明

#### 单选题
- 学生点击选项后立即显示对错
- 绿色表示正确，红色表示错误
- 可设置解析，点击"查看解析"按钮显示

#### 多选题
- 学生可勾选多个选项
- 点击"提交答案"后批改
- 正确选项显示绿色，错误选项显示红色

#### 填空题
- 可设置多个空格
- 学生输入后失焦自动判断
- 支持大小写敏感设置

#### 问答题
- 开放式文本输入
- 可设置字数限制
- 学生可查看参考答案

#### 盲盒
- 随机或顺序抽取
- 点击按钮触发翻转动画
- 适合课堂互动

#### 嵌入HTML
- 可粘贴任意HTML代码
- 在iframe沙箱中安全渲染
- 适合嵌入外部资源或自定义交互

## 设计原则

本项目严格遵循现代化的UI/UX设计规范：

- **极简美学**: 干净、专注、无多余装饰
- **5级灰度系统**: 精确控制视觉层次
- **圆角系统**: 胶囊形按钮、12px卡片、8px容器、4px输入框
- **4px倍数间距**: 保证布局节奏感
- **高对比度**: 确保所有文本可读性达到WCAG AA标准
- **键盘友好**: 所有交互元素支持键盘导航
- **响应式**: 完美适配移动端

## 生成的HTML特性

生成的HTML文件具有以下特点：

- ✅ **完全独立**: 所有CSS和JavaScript内联，无外部依赖
- ✅ **离线可用**: 无需网络连接即可使用
- ✅ **跨平台**: 在任何现代浏览器中都能正常运行
- ✅ **美观现代**: 采用现代化的设计风格
- ✅ **交互完整**: 所有题型的判分和反馈逻辑都已实现
- ✅ **响应式**: 自适应手机、平板、电脑屏幕

## 注意事项

- 首次使用需要在Supabase控制台中手动创建数据库表
- 生成的HTML文件包含学生答题交互，但不会回传数据到服务器
- 编辑器会自动保存，建议重要内容手动点击保存按钮
- 盲盒的随机/顺序模式在每次刷新页面后重置
